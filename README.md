## 🚀 Getting Started

voxagent is a voice-powered terminal app for Windows. It lets you speak a command, hear a response, and keep working without typing much. It runs fully offline, so your voice data stays on your computer.

Use it when you want to:
- start apps
- ask quick questions
- run terminal commands
- work hands free
- keep your data local

## 💻 What You Need

Before you install voxagent, make sure your PC has:
- Windows 10 or Windows 11
- a working microphone
- speakers or headphones
- enough free space for the app and voice files
- a stable local setup for offline use

For best results, use:
- a headset mic
- a quiet room
- Windows audio input set to the right microphone

## 🧭 How to Use

voxagent listens for your voice, turns speech into text, then sends the text to its local agent logic. It can answer simple questions and help with terminal tasks.

A normal use flow looks like this:
1. Start the app.
2. Speak a command.
3. Wait for the app to process your words.
4. Read or hear the answer.
5. Speak another command if needed.

Keep your commands short at first. Clear phrases work best.

## ⚙️ Offline Use

voxagent is built for offline use. That means it can work without sending your voice to a cloud service. This helps when you:
- want local processing
- have limited internet
- need a private voice setup
- work in a place with no connection

For offline mode, keep the local voice and agent files on your PC. Do not remove the app data it needs to run.

## 🔊 Voice Tips

To get better results:
- speak one command at a time
- pause for a second after speaking
- use plain words
- avoid background noise
- stay close to the mic
- keep the mic level at a normal range

If the app misses words:
- check the selected microphone
- speak more clearly
- move closer to the mic
- close other apps using the mic

## 🛠️ Basic Troubleshooting

If voxagent does not start:
- confirm the file finished downloading
- extract the zip file first
- run the app from the extracted folder
- check that Windows did not block the file

If the microphone does not work:
- open Windows Sound settings
- select the right input device
- test the mic in another app
- unplug and reconnect a USB mic if you use one

If voice input feels slow:
- close other heavy apps
- use a faster PC if possible
- keep the app in a local folder on your main drive

If you get no response:
- make sure the app stays open
- check your speaker volume
- try a short test command
- restart the app
